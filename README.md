# BlackJack

## WHAT IS IT

A CommandLine BlackJack Game written in C++

## RUN ON YOUR LOCAL MACHINE

```bash
# clone the repository
git clone https://github.com/flaviaouyang/blackjack-cpp.git

# navigate the directory
cd blackjack-cpp

# compile
g++ main.cpp -o main
# or
g++ main.cpp

# run
./main
# or
./a.out

# have fun;)
```
